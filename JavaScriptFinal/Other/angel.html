Scripting Languages I Alternate Final

One Site X 3

Your FINAL assignment for the Scripting Languages I course is to create an HTML file using this page content, that will be styled differently three times, using one of three different CSS style sheets that you will create.

This base XHTML page is built using the XHTML STRICT DTD. There is an alternate base page for those of you who would prefer to use the HTML5 DTD.

If you like, you are allowed to add the following elements (and NO others):

  • div
  • span
  • p
  • h1 h2 h3 h4 h5 h6
  • ul ol li
  • dl dt dd
  • img
  • br
  • hr
  • a

Done looking then click here to go back Portfolio

INSTRUCTIONS:

Start by altering this HTML file by inserting DIV, SPAN, and (optionally) allowed elements, using them to help definine the structure of your page. Use class and id to selectively detail page elements, and save the CSS to an external CSS file that has been linked (using the link element) to this HTML file. None of the content of your HTML files can be re-arranged (within the HTML page itself). ALL placement, positioning, coloring, etc. MUST be done using that page's linked stylesheet.

When you've finished your first styled page, copy the HTML file, and make two more versions of that EXACT page (the only things that should change from one HTML page to the second and third pages will be the name of the CSS file that you are linking to that is styling the HTML page, and the design number that you'll change in the title tags). We highly recommend you name your files so that they can be easily identified, such as:

  • final1.html --> which will link to --> final1.css
  • final2.html --> which will link to --> final2.css
  • final3.html --> which will link to --> final3.css

When you are finished, VALIDATE all of your HTML and CSS. When they all pass validation (strict), zip all of your files up (you should have 3 .html and 3 .css files) and attached the zipped file in Canvas. If your styles include images and/or background images, be sure to include them when you zip your project up. Remember -- All three of your HTML files must be IDENTICAL, with the exception of the href value in the link element that determines which style sheet is being used. If you have any questions, be sure to contact an instructor.
